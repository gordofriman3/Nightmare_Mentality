# Nightmare_Mentality
Acá se encuentra el juego "Nightmare Mentality" un juego de pagina interactiva y survival horror, y se encuentra finalizado. Este juego esta inspirado en juegos de terror como Resident Evil, Silent hill, Cry of Fear, pero de una forma mas simplificada y liviana. Como creador, confirmo que tengo en plan hacerle un remake en 3D.
